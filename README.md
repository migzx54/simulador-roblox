# Simulador Roblox

Projeto de simulador criado no Roblox Studio usando Lua.

O jogo é focado em progressão, onde o jogador ganha força ao clicar, coleta moedas e evolui ao longo do tempo.

## Funcionalidades
- Sistema de força
- Sistema de moedas
- Interface mostrando status do jogador
- Código organizado em Lua

## Tecnologias
- Lua
- Roblox Studio

## Objetivo
Projeto criado para aprendizado e prática de desenvolvimento de jogos no Roblox.

## Observação
Algumas ideias podem ter sido inspiradas em outros simuladores, usadas apenas como referência.
